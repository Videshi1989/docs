Fetch(Clone) project by git hub repository:

	Step 1: Open vs code,By extension icon search "GitHub Pull Requests" OR "GitHub Pull Requests and Issues" and install it.
	
	Step 2: Open any one github repository >> code (dropdown) >> copy the url by icon.
	
	Step 3: Click on Setting icon in vs code, select "Command Palette(ctrl+shift+p)", search and select "Git: Clone" then paste the github    repository url then press Enter.
	
	Step 4: We will create a folder(any name) on any location where we want tu put then click on "Select Repository Location".
			it will take few minute to pull then click on "Open"
			
	Step 5: Run the below command:
			npm install
